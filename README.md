# Visual Foundry Website

## Overview

Visual Foundry is a responsive multi-page website designed and developed as part of my Bachelor of Information Technology.

The project focused on creating a professional portfolio website for a fictional interaction design company while applying 
responsive web design principles, UI/UX design, accessibility, and clean HTML/CSS practices.

---

## Features

- Responsive layout
- Multi-page navigation
- Consistent visual styling
- Custom branding
- Image optimisation
- Accessible navigation

---

## Technologies Used

- HTML5
- CSS3
- Visual Studio 2022
- Figma
- Adobe Photoshop

---

## What I Learned

This project strengthened my understanding of:

- Responsive web design
- HTML page structure
- CSS layouts and styling
- Navigation design
- UI and UX principles
- Accessibility considerations
